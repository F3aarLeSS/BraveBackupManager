# Brave Backup Manager

Portable PowerShell utility to backup and restore Brave Browser.

## Features

✔ Backup Brave Stable
✔ Backup Brave Beta
✔ Backup Brave Dev
✔ Backup Brave Nightly
✔ Restore to any installed Brave edition
✔ Multi-profile support
✔ Automatic backup verification
✔ Metadata support

## Requirements

Windows 10/11
PowerShell 5.1+

## Quick Start

```powershell
irm https://f3aarless.github.io/BraveBackupManager/install.ps1 | iex
```
